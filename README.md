# Duck Game Music Manager
Simple Manager of Sound Packs for Duck Game

## How to Use It
1. Download latest [release](https://github.com/ryze312/Duck-Game-Music-Manager/releases)
2. Create a folder near DGMM.exe called Music Packs
3. Copy your Music Packs inside this folder
4. Install it via manager

## How to Create Packs
1. Create file structure like this <br />

![Structure](Images/Structure.svg?raw=true "File Structure")

2. Place files inside:
   * **SFX** to change sound effects (.xnb)
   * **Music** to change misc music (.ogg)
   * **InGame** to change music played in multiplayer (.ogg)
