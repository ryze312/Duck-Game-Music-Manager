# Duck Game Music Manager
Простой менеджер звуковых паков для Duck Game

## Использование
1. Загрузите последний [релиз](https://github.com/ryze312/Duck-Game-Music-Manager/releases)
2. Создайте папку рядом с DGMM.exe под названием Music Packs
3. Скопируйте паки в папку Music Packs
4. Установите их с помощью менеджера

## Созданиe паков
1. Создайте файловую структуру <br />

![Структура](Images/Structure_RUS.svg?raw=true "Файловая структура")

2. Положите файлы в:
   * **SFX**, чтобы заменить звуковые эффекты (.xnb)
   * **Music**, чтобы поменять специальную музыку (.ogg)
   * **InGame**, чтобы поменять музыку, играющую в мультиплеере (.ogg)
